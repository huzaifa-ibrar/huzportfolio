# Huzaifa Ibrar - Enhanced Portfolio Website

A luxurious, modern portfolio website with gold, black, and white aesthetic inspired by your profile picture.

## 🎨 New Features & Enhancements

### Visual Design
- **Color Scheme**: Elegant gold (#D4AF37, #FFD700), black (#0F0F0F), and white theme
- **Your Profile Picture**: Prominently displayed in the About section with a glowing animated border
- **Gradient Backgrounds**: Sophisticated radial and linear gradients throughout
- **Gold Accents**: Luxurious gold highlights on buttons, borders, and text

### Animations & Effects

#### 1. **Cursor Trail Effect**
   - Glowing golden particle trail follows your cursor
   - Smooth, elegant animation that adds sophistication
   - Automatically disabled on mobile devices

#### 2. **Typing Effect**
   - Hero title types out letter by letter on page load
   - Creates an engaging first impression

#### 3. **Parallax Scrolling**
   - Hero content moves at different speeds while scrolling
   - Adds depth to the design

#### 4. **3D Tilt Effect on Project Cards**
   - Project cards tilt based on mouse position
   - Creates an interactive, premium feel

#### 5. **Hover Animations**
   - Buttons glow and lift on hover
   - Cards transform and reveal gradients
   - Skill icons rotate 360°
   - Social links bounce up

#### 6. **Scroll Animations**
   - Elements fade in and slide up as you scroll
   - Staggered animation delays for multiple items
   - Smooth, professional transitions

#### 7. **Timeline Pulse**
   - Experience timeline markers pulse with golden glow
   - Creates visual interest and draws attention

#### 8. **Floating Effect**
   - Contact items gently float up and down
   - Adds subtle motion to the page

#### 9. **Shimmer Effect**
   - Golden text shimmers and brightens
   - Applied to highlighted name and titles

#### 10. **Grid Background Animation**
   - Subtle moving grid pattern in hero section
   - Adds technical, modern feel

### New Content

#### **Fragment Storage Microservice Project**
   - Added as the featured first project
   - Complete with tech stack badges (Node.js, Express, Docker, AWS ECS, S3, DynamoDB, Cognito)
   - Detailed description of features and achievements

### Layout Improvements

#### **About Section**
   - Your professional photo with animated golden border
   - Split layout: image on left, text on right
   - Skill categories now have icons (code, layers, database, cloud)
   - Enhanced card hover effects with light sweep animation

#### **Projects Section**
   - **2-column grid layout** as requested
   - Large project numbers (01, 02, 03, 04) as background
   - Enhanced hover effects with 3D tilt
   - More prominent tags with hover animations

#### **Experience Section**
   - Glowing golden timeline
   - Animated markers that pulse
   - Enhanced card hover with golden border reveal

## 🚀 Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Advanced animations, gradients, transforms
- **Vanilla JavaScript**: Canvas API for cursor trail, Intersection Observer for scroll animations
- **Font Awesome**: Professional icons

## 🎯 Performance Features

- **Optimized Animations**: GPU-accelerated transforms
- **Lazy Loading**: Elements animate only when visible
- **Responsive Design**: Mobile-optimized (cursor trail disabled on mobile)
- **Smooth Scrolling**: Native smooth scroll behavior

## 📱 Responsive Breakpoints

- **Desktop**: Full experience with all animations
- **Tablet (1024px)**: Single column about section, maintained animations
- **Mobile (768px)**: Hamburger menu, simplified layout, cursor trail disabled
- **Small Mobile (480px)**: Optimized typography and spacing
