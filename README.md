# Huzaifa Ibrar - Portfolio Website

A modern, luxurious portfolio website with a sleek side navigation design featuring gold, black, and white aesthetics.

## 🎨 Design Highlights

### **New Modern Layout**
- **Side Navigation Bar**: Elegant fixed sidebar with icon-based navigation
- **Split Hero Section**: Your professional photo alongside introduction
- **Card-Based Design**: Modern card layouts for experiences and projects
- **Responsive Grid System**: Clean 2-column layouts for content

### **Color Scheme**
Inspired by your professional photo:
- **Primary Gold**: #D4AF37 - Main accent color
- **Accent Gold**: #FFD700 - Bright highlights
- **Deep Black**: #0F0F0F - Sophisticated background
- **Pure White**: #FFFFFF - Text and contrast

## ✨ Features

### Interactive Elements
1. **Cursor Trail Effect** - Golden particle trail follows your cursor
2. **3D Card Tilts** - Project cards respond to mouse movement
3. **Smooth Scroll Progress** - Top progress bar shows scroll position
4. **Animated Statistics** - Numbers count up when scrolled into view
5. **Ripple Button Effects** - Interactive button click animations
6. **Parallax Scrolling** - Hero elements move at different speeds
7. **Hover Transformations** - Cards lift and glow on hover
8. **Staggered Animations** - Skills appear one by one
